Este repositorio contiene el código fuente de mi sitio web personal y portafolio profesional. Es una página web de una sola vista (Single Page Application - SPA) diseñada para destacar mi perfil, habilidades técnicas, proyectos y medios de contacto.

## 🚀 Características y Novedades

*   **Diseño Moderno y Atractivo:** Interfaz de usuario "Dark Theme" con colores de acento vibrantes (verde neón), uso de transparencias (glassmorphism) y animaciones fluidas (hover effects y transiciones).
*   **Regla Global de Legibilidad:** Implementación rigurosa de contraste (Fondo `#2D2D44` y Textos `#CBD5E1`) para garantizar que la información en todas las tarjetas sea perfectamente legible bajo el tema oscuro.
*   **Línea de Tiempo (Timeline):** La sección de Estudios y Formación cuenta con un diseño de tarjetas apiladas conectadas por una línea de tiempo vertical visual, ordenadas desde lo más reciente.
*   **Totalmente Responsivo:** Adaptable a todo tipo de pantallas (móviles, tablets, escritorio) gracias a media queries personalizadas y utilidades de Bootstrap 5.
*   **Navegación Fluida:** Scroll suave (smooth scrolling) a las diferentes secciones de la página desde una barra de navegación fija (sticky navbar).

## 🗂️ Secciones Integradas

*   **Inicio (Hero):** Presentación personal y resumen rápido.
*   **Estudios:** Timeline visual con Formación Universitaria (Ingeniería Informática), Certificación Oficial MinTIC (Bootcamp Talento Tech) y nivel de Idiomas.
*   **Habilidades:** Tecnologías y lenguajes manejados (HTML, CSS, JS, Python, SQL).
*   **Proyectos:** Vitrina interactiva para mostrar futuros desarrollos y proyectos actuales.
*   **Intereses:** Áreas de interés dentro del rubro tecnológico (Frontend, Backend, Bases de Datos).
*   **Contacto:** Llamados a la acción directos para conectar vía correo electrónico o GitHub.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica de la página.
*   **CSS3 (Vanilla):** Estilos personalizados, variables CSS globales para el manejo del tema oscuro y legibilidad, pseudo-elementos para el Timeline.
*   **Bootstrap 5:** Framework CSS para el sistema de grillas (Grid) y estructuración responsiva.
*   **JavaScript (Vanilla):** Interactividad adicional, control del menú en móviles y efectos en la barra de navegación.
*   **Font Awesome 6:** Iconografía vectorial a lo largo de todo el sitio.
*   **Google Fonts:** Tipografía principal "Poppins" para un aspecto moderno y legible.

## 📂 Estructura del Proyecto

El proyecto está contenido principalmente en un único archivo, lo que facilita su lectura e implementación rápida.

```text
portafolio/
│
└── index.html      # Archivo principal que contiene la estructura (HTML), estilos (CSS) y lógica (JS)
```

## 💻 Instalación y Uso

Dado que el proyecto utiliza tecnologías web estándar del lado del cliente, no requiere de una instalación compleja o configuración de servidores backend.

1.  Clona este repositorio o descarga los archivos.
    ```bash
    git clone https://github.com/sebas05di/portafolio.git
    ```
2.  Navega a la carpeta del proyecto.
3.  Abre el archivo `index.html` en tu navegador web preferido (Chrome, Firefox, Edge, Safari, etc.).
